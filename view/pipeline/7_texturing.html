Texturing

The objective of this step is to texture the generated mesh.

If the mesh has no associated UV, it computes automatic UV maps. AliceVision implements a basic UV mapping approach to minimize the texture space. The standard UV mapping approach is provided by [Levy2002].

The 3D texturing is based on a generalization of the multi-band blending described in [Burt1983]. For each triangle, we use the visibility information associated to each vertex to retrieve the texture candidates. We filter the cameras without a good angle to the surface to favor fronto-parallel cameras and finally average the pixel values. This approach is in the same spirit than [Baumberg2002] and [Allene2008].

More advanced methods can be added to refine the alignment before averaging pixel colors.
